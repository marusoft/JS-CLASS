## Data Types

### String data type
```javascript
console.log("I am learning js");
console.log("20");
```

### Number Data types

```javascript
console.log(23);
console.log(55);
```
### Boolean Data types

```javascript
console.log(true);
console.log(false);
```
### Null Data types

```javascript
null
var myname = null;
```
### Undefined Data types
undefined
```javascript
var myname;
```

### Object Data types
```javascript
var myProfile = {
  name: "alimi",
  age: 23,
  classYear: "200L",
  address: "3, adeyemo close"
};
```
